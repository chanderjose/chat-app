## 🚀 How to Run the Project

Getting this up and running on your local machine is super easy. Just follow these steps:

1. **Start the backend server:** Before running the frontend, make sure the backend is running! It lives in the `../backend` folder. You'll need that running so the frontend has something to connect to for the real-time websocket magic.
2. **Install dependencies:** Open a terminal in this directory (`frontend`) and run:
   ```bash
   npm install
   ```
3. **Start the development server:** Once everything is installed, fire it up with:
   ```bash
   npm run dev
   ```
4. **Open it in your browser:** Vite will typically give you a local URL (like `http://localhost:5173`). Just click that or paste it into your browser, and you're good to go!
